# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project focuses on predicting house prices using the Linear Regression algorithm. The objective is to build a machine learning model that estimates house prices based on various property features such as area, number of bedrooms, bathrooms, parking spaces, and other relevant factors.

Linear Regression is one of the most fundamental supervised machine learning algorithms used for predicting continuous numerical values. This project demonstrates the complete machine learning workflow, from data preprocessing to model evaluation and visualization.

---

## 🎯 Aim
To develop a Linear Regression model that can accurately predict house prices using property-related features and evaluate its performance using regression metrics.

---

## 📂 Dataset
**Dataset Link:** *(https://www.kaggle.com/code/ashydv/housing-price-prediction-linear-regression)*

The dataset contains:
- Numerical and categorical house features
- Target variable: House Price

Example features:
- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Furnishing Status
